Bases terico-matematicas y Funcionamiento

El Gradient Boosting funciona mediante la creacin de rboles de decisin secuenciales, donde cada rbol se enfoca en corregir los errores del modelo anterior1. En la primera iteracin, se crea un rbol de decisin que minimiza los errores del modelo inicial. Luego, en cada iteracin subsiguiente, se crea un nuevo rbol que se enfoca en los residuos (es decir, los errores) del modelo anterior. Este proceso se repite hasta que se alcanza un nmero predefinido de rboles (especificado por el parmetro n.trees en la funcin gbm()) o hasta que los errores ya no se pueden reducir significativamente.

Cada rbol se construye utilizando un subconjunto de los datos (seleccionado aleatoriamente con reemplazo), lo que ayuda a mejorar la robustez del modelo y reduce la varianza. Adems, en cada iteracin, se aplica una tasa de aprendizaje (especificada por el parmetro shrinkage en la funcin gbm()) que controla cunto contribuye cada rbol al modelo final. Esto ayuda a prevenir el sobreajuste al poner ms peso en los rboles que mejoran ms el modelo.

df_compare <- data.frame(
  Quality_Real = df.Wine.Test$quality,
  Quality_Predicted_Model1 = mdl.Predict,
  Quality_Predicted_Model2 = mdl.Predict2,

  geom_point(color = "darkblue") +
  geom_abline(intercept = 0, slope = 1, color = "darkred") +
  labs(title = "Modelo 1", x = "Calidad Real", y = "Calidad Predicha")
